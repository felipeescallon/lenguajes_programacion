CHAT REAL SI SOLO SE TRABAJA CON CLIENTES Y SERVIDOR
LO DE LOS LIDERES NO ESTA EN LA VIDA REAL. SOLO FUE REQUERIMIENTO DEL PROFE

OJO:
TODOS LOS PROGRAMAS EMPIEZAN A EJECUTARSE POR EL MAIN =>
	SERVIDOR: MAIN EN InterfazServidor.java
	CLIENTE: MAIN EN InterfazdeChat.java

SE NECESITA:

*UN SERVIDOR, VARIOS CLIENTES.

*SOLO SE NECESITA LA DIRECCION IP DEL SERVIDOR(el programa servidor esta como "localhost")
 YA Q ESTE ENCUENTRA LA DE LOS 
 CLIENTES Q SE CONECTEN A EL.
 POR EJP: LA IP DE ESTE PC ES: "192.168.0.74". SE LA HALLA ASI:

	1.CLICAR EN ICONO DE RED LAN (PARTE INFERIOR DERECHA DE LA PANTALLA)
	2.ELEGIR TCP/IP
	3.PROPIEDADES

*ENTONCES SE DEBEN MODIFICAR LAS SIGUIENTES CLASES EN CADA CLIENTE:

- InterfazdeChat.java
- InterfazdeDatos.java
- TelecomunicacionTX.java
	EN new Socket(String destino,int puerto) se debe modificar el destino por
	la IP a la q se conectaran (la hallada para el servidor):

	destino = "xxx.xxx.xxx.xxx"	


*DEBE CONFIGURARSE EL ODBC PARA EL MANEJO DE LA BASE DE DATOS POR PARTE DEL SERVIDOR:
 -AQUI HEMOS LLAMADO AL NOMBRE LOGICO COMO Modificacion.Si no existe, entonces hay que crearlo;
  y si existe, entonces solo se debe configurarlo para que el acceso a la base de datros que se 
  va a manejar sea  coreecto. Asi:

---Crearlo:	
	1.PANEL DE CONTROL
	2.HERRAMIENTAS ADMINISTRATIVAS
	3.ODBC
	4.AGREGAR
	5.MICROSOFT ACCESS DRIVER(*.mdb)
	6.DARLE EL NOMBRE LOGICO
	7.SELECCIONAR
	8.ENRUTAR HACIA DONDE ESTE LA BASE DE DATOS (DEBE ESTAR DENTRO DE LA CARPETA SERVIDOR)
	9.aceptar

---Configurarlo:
	1.PANEL DE CONTROL
	2.HERRAMIENTAS ADMINISTRATIVAS
	3.ODBC
	4.SELECCIONAR Modificacion
	5.SELECCIONAR
	6.ENRUTAR HACIA DONDE ESTE LA BASE DE DATOS (DEBE ESTAR DENTRO DE LA CARPETA SERVIDOR)
	7.aceptar
   	
*PRIMERO DEBE EJECUTARSE EL SERVIDOR, EL CUAL DEBE SER PERMANENTE, Y LUEGO LOS CLIENTES.

*SI ALGO FALLA, ENTONCES DAR Ctr+C EN LA PANTALLA DEL C:\(SIMBOLO DEL SISTEMA)
