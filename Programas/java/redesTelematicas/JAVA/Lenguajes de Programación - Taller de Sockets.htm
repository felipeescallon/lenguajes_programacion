TALLER: SOCKETS EN JAVA

Diseñe dos programas que muestren una aplicación cliente/servidor haciendo uso de sockets de flujo. La aplicación cliente debe conectarse con el servidor, la aplicación envía datos al cliente y éste exhibe los datos recibidos.

Clases necesarias para compilar el programa:

import java.io.*;
import java.net.*;

Apertura de Sockets:

Cliente 
Socket cliente;
try {
  cliente = 
     new Socket("maquina", puerto);
}
catch (IOException e) {
  System.out.println(e);
}
Servidor 
ServerSocket servidor;
Socket conexion;
try {
  servidor = new ServerSocket(9090);
  conexion = servidor.accept();
  //...
}
catch (IOException e) {
  System.out.println(e);
}

Cierre de Sockets:

Cliente Servidor 
cliente.close();
 
conexion.close();
